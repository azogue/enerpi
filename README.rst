⚡⚡ ︎ENERPI AC Current Meter for Raspberry PI ⚡⚡
===============================================

A simple current meter based on:
- SCT-030 030 Current sensor,
- MCP3008 Analog to Digital converter (on RASP.IO Analog Zero Hat)
- Raspberry PI

----

To get started:

.. code:: bash

    python3 -m pip install enerpi
    enerpi -h
